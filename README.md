## 🔖 Sobre o Projeto

- Plataforma de pesquisa dedicada ao mundo cinematográfico! Neste projeto, cada card oferece um mergulho profundo em filmes e documentários, apresentando detalhes, como descrições e datas de lançamento. Se você é um amante da sétima arte ou está em busca de novas e emocionantes descobertas, esta plataforma foi feita sob medida para você.

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- JavaScript
